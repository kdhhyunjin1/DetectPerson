사용방법

0. 사전 세팅이 필요함.
"MSCVP140D.dll이 없어 코드 실행을 진행할 수 없습니다."에러가 나타날 경우 Visual Studio가 설치. opencv가 설치가 안되어있어도 실행은 될 것으로 보임.
(노트북으로 개발했는데 opencv가 없는 데스크탑에서 잘 실행됨. 단, 다른 사람들에게 실행했을 때 위의 에러가 발생함. 찾아보니 visual studio가 없어서 생긴 문제라고 함.)

1. 사람이 등장하는 mp4파일을 Video폴더 안에 넣는다. 기본적으로 test.mp4 파일 첨부.

2. DetectPerson.exe파일을 실행 후 파일의 이름을 입력한다.

3. 동영상이 나오면서 사람 주변에 사각형이 쳐져있는 것을 확인.

4. esc를 누르면 사람이 있었던 곳에 조그만 사각형이 생겨있음. 사람이 오래 있는 곳일 수록 짙은빨강->노랑->하양에 가까워짐.

5. esc를 한번 더 누르면 사람들이 이동했던 경로가 나타남.

6. esc를 한번 더 누르면 종료.




※ Nonexistent File이 나왔다면 파일 이름이 올바른지 확인. 특히, 확장자까지 정확히 입력했는지 확인.
